Get-GPLinks

SYNOPSIS

Script to provide either a TXT or CSV report of linking relationships between OUs and GPOs.

SYNTAX

Both

Get-GPLinks [[-Path] <String>] [-BothReports] [<CommonParameters>]

FullReport

Get-GPLinks [-Path] <String> [-AllOUs] [-RootOnly] [<CommonParameters>]

CSVReport

Get-GPLinks [-Path] <String> [-CSVReport] [<CommonParameters>]

DESCRIPTION

Script that reports what OUs in the domain have what GPOs linked to them. Intended for use with Group Policy backup as exported policies do not retain the link information.

EXAMPLES

EXAMPLE 1

Get-GPLinks -Path "C:\Temp"

PARAMETERS

Output path for files to be saved. This should be a directory path, not a file path.
